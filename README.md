# Todo App

This Todo App built with [Vite](https://vitejs.dev/) + [React](https://reactjs.org/) that stores your tasks.

### See Live : [Here](https://todoapp-redux-react.netlify.app)

## Project Details

- MongoDB
- Express.js
- React.js
- Node.js
- Redux Toolkit
- Axios
- Bootstrap
- Ant Design
- SASS
